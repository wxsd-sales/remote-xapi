# Remote xAPI

This is a example JavaScript macro library which simplifies making xAPI requests to multiple remote Cisco Collaboration Devices. The library has a request queuing system to avoid exhausing available HTTP Client connections and convienently returns results in a JSON format for easy result processing within Cisco Collaboration Device macro JavaScript runtime.
